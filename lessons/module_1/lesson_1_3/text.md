<b>💻 Что такое программа?</b>

Компьютерная <b>программа</b> — это список команд (инструкций) для компьютера. Команды могут быть любыми, например:

• считать информацию с клавиатуры;
• произвести арифметические вычисления (+, −, *, /);
• вывести информацию на экран.

В каждом компьютере установлено много разнообразных программ:

• <b>Google Chrome</b> — программа-браузер для просмотра сайтов;
• <b>Telegram</b> — звонки и мгновенные сообщения;
• <b>Windows, macOS, Linux</b> — тоже программы (операционные системы).

Для создания программ используются <b>языки программирования</b>. Выбор языка обычно зависит от задачи, которую нужно решить.

━━━━━━━━━━━

<b>📖 Язык программирования</b>

<b>Язык программирования</b> — набор правил, по которым компьютер понимает команды и выполняет их.

Текст программы на любом языке называется <b>программным кодом</b>.

<b>Компилируемые языки</b> (C, C++, Pascal):
программу сначала полностью проверяют на ошибки, затем переводят в <b>машинный код</b>. Это делает <b>компилятор</b>.

<b>Интерпретируемые языки</b> (Python, PHP, Ruby):
код не переводится целиком в машинный код. <b>Интерпретатор</b> читает программу построчно и выполняет каждую команду.

<b>Гибридный подход</b> (C#, Java):
код компилируется в промежуточный <b>байт-код</b>, а при запуске уже переводится в машинный код.

<b>🌟 Язык Python</b>

Python создал голландский программист <b>Гвидо ван Россум (Guido van Rossum)</b> в 1991 году.

Название — от британского комедийного сериала <i>Monty Python's Flying Circus</i>.

<b>⚖️ Python 2 vs Python 3</b>

Основные версии — <b>Python 2</b> и <b>Python 3</b>.

• Python 2 — устаревающая ветка;
• Python 3 — современная.

Почему не отказались от Python 2? У версии 3 нет полной обратной совместимости, а на Python 2 написано очень много программ — переписать всё сразу невозможно.

<b>В этом курсе — только Python 3.</b> О Python 2 говорить не будем.

<b>✅ Преимущества Python</b>

• <b>Интерпретируемый</b> — не нужна отдельная компиляция, запуск прямо из исходного кода
• <b>Высокоуровневый</b> — код ближе к человеческому языку
• <b>Платформонезависимый</b> — пишем и запускаем на Linux, Windows, macOS
• <b>Open source</b> — открытый и бесплатный
• <b>Простой</b> — легко начать обучение
• <b>Встраиваемый</b> — можно встроить в другие системы
• <b>Динамический</b> — проще писать небольшие программы
• <b>Огромные библиотеки</b> — готовые решения «на любой вкус»
